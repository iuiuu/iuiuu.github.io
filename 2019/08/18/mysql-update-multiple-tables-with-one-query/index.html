表A（test_a）：

id user_id dept_id update_time
1 34242 313
2 35322 320
3 35639 321

表B（test_b）：

id user_id patient_id dept_id create_time
1 34242 342 0 2018-05-23 09:33:45
2 35322 365 0 2018-05-24 00:13:05
3 35639 398 0 2018-05-24 00:43:18
