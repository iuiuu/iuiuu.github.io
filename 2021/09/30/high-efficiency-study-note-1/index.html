模块一：底层思维

Created2021-09-30|Updated2021-11-13|读书学习笔记
|Post View:
  • 模块一：底层思维
  • 模块二：高效学习
  • 模块三：高度自律
  • 模块四：时间管理
  • 模块五：学霸养成

01｜如何减少对学习的排斥和厌恶心理，使其变得相对愉悦？

  1. 在学习的过程中，可以穿插安排喜欢且擅长的环节，而不是一直死磕艰难的任务。
  2. 在制定计划的时候，量力而行，不要制定完不成的计划；
  3. 详细记录当天具体的时间使用情况，把自己的学习成果具像化，借以获得正反馈和成就感。
  4. 在感到疲倦时，可以试试换个位置或者换个场所。
  5. 每天别给自己安排太多类型的任务。
  6. 学习的时候，不用某种标准框住自己，也别总对自己的表现评头论足，更不必急着追求某种结果，而是注重做事的过程，把努力当成一种习惯。

02｜学会这4点，你也可以告别伪勤奋

伪勤奋的做法：

  1. 花大量时间做容易的环节；
  2. 从来不去检测自己获得了什么；
  3. 只是埋头苦学，从来不总结规律；
  4. 道德许可效应让你原地徘徊，沉迷于已经努力的幻觉之中。

相应解决思路：

  1. 制定合理的目标计划，适当安排有挑战的任务（需要思考、需要克服困难的任务）；
  2. 及时获得反馈，及时检测所学东西，是否记住，是否会做题；
  3. 观察、分析、总结自己的学习方法，对方法进行针对性的调整和完善，并且有意识地指导自己接下来的努力方向；
  4. 将想做的事情最简化，不搞形式主义，直接开始。

03｜掌握考试思维：成为一个特别会考试的人

  1. 备考时，要学会用考试的思维去学习（考什么学什么）；
  2. 研究考试题（出题范围和出题思路），分析题目要求你怎样回忆知识点、怎样运用知识点。并据此决定备考时如何学以及学什么；
  3. 最好能找到考试题的答案，知道高分的答案具有哪些元素，需要满足怎样的标准；
  4. 学习时，做到即用即学和即学即用，能对知识点有透彻的理解和记忆，并且在考试中，做到更为高效准确地调取，以及运用。
  5. “检索式练习”，通过提问和主动回想的方式，去巩固知识点。

04｜用科学原理告诉你如何在考场上超常发挥

1、在解决问题的时候，有一个用于暂时存放信息和加工信息的记忆系统，叫做“工作记忆”，它的容量有限。工作记忆能够存储和处理的信息容量越多，意味着解决问题的能力就越强。

2、考试中发挥失常，是因为过于担忧考试结果，纠结自己是不是做错了，以至于，相当一部分工作记忆的容量被焦虑占用。而真正用于答题的记忆容量，非常有限。

3、要想超常发挥，重要的是，考试时不去预先给自己设置标准，不去评价和思考自己做得怎么样。做完这一题，再做下一题；做下一题时，忘记上一题。

4、在平时的学习中，我们常常怀疑自己的方法是否正确，占用了很多的工作记忆，影响学习效率。有效的解决方法是：给自己一个小时的限定时间，先用眼下的方法，尽量专注学习。如果过程中对学习方法有更好的想法，就先记下来，一小时结束后，花两分钟对学习方法进行调整。
