小结

判断 Python 集合是否为空有以下四种方法:

  1. 使用 len() 方法

如果 len(s) 返回 0，则表示集合为空。

  1. 使用 not 关键字

如果集合为空，not s 返回 True，否则返回 False。

  1. 使用 bool() 方法

bool(s) 将集合转换为布尔值，如果为空集合，返回 False，否则返回 True。

  1. 与空集合进行比较
