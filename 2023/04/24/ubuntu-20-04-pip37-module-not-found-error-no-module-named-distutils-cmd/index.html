要解决这个错误，你需要安装 distutils 模块，一般可以直接通过 apt-get 命令安装 python3-distutils 包来解决缺失问题。从你安装该模块的信息可以看到，你已经安装了该包，只不过该包的版本并没有跟你现在 Python 的版本对应上。
