Lambda 表达式使用 -> 来表示：

大多数 SQL 表达式都可以在 Lambda 主体中使用，但有几个例外：

  • 不支持子查询。如 x -> 2 + (SELECT 3)
  • 不支持聚合。如 x -> max(y)
