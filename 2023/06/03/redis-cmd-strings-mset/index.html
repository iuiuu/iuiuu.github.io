时间复杂度

$O(N)$

其中 N 是要设置的 key 的数量。

ACL类别

@write, @string, @slow

将给定的键设置为它们各自的值。MSET 用新的值替换现有的值，就像普通的 SET 一样。如果你不想覆盖现有的值，请参阅 MSETNX。

MSET 是原子性的，所以所有给定的键都是一次性设置的。也就是说，客户端不可能看到一些键被更新而另一些键没有变化。
