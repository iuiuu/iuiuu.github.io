Redis STRLEN 命令

Created2023-06-04|Updated2023-06-06|Redis教程
|Post Views:

语法

可用版本

≥ 2.2.0

时间复杂度

$O(1)$

ACL类别

@read, @string, @fast

返回存储在 key 处的字符串值的长度。当 key 存储的值不是字符串类型时，则返回错误。

返回值

返回一个整数：

  • key 对应值的字符串长度；
  • 如果 key 不存在，则返回 0。
