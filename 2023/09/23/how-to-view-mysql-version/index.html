方法三：通过 MySQL 管理工具

如果你使用了一些图形化界面的 MySQL 管理工具，比如 phpMyAdmin、DBeaver 或 Navicat，则可以通过以下步骤查看 MySQL 版本：

  1. 打开 MySQL 管理工具并连接到数据库实例。
  2. 导航到关于或系统信息等类似选项。
  3. 在相应的选项卡或视图中，可以看到 MySQL 版本信息。
