Available Commands

  • serve: Start the Ollama service.
  • create: Create a model using a Modelfile.
  • show: Display detailed information about a model.
  • run: Run a model.
  • stop: Stop a running model.
  • pull: Download a model from a registry.
  • push: Upload a model to a registry.
  • list: List all available models.
  • ps: List currently running models.
  • cp: Copy a model.
  • rm: Delete a model.
  • help: Get help for any command.

Flags

  • -h, --help: Show help information for Ollama.
  • -v, --version: Display the version of Ollama.
