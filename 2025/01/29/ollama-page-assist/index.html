Ollama Page Assist

Created2025-01-29|Updated2025-02-18|Ollama Tutorial
|Post Views:

Page Assist is an open-source browser extension that provides an intuitive interface for interacting with local AI models. It allows users to chat and engage with local AI models directly on any webpage.

Key Features

  • Sidebar Interaction: Open a sidebar on any webpage to chat with your local AI model and get intelligent assistance related to the page content.
  • Web UI: A ChatGPT-like interface for more comprehensive conversations with the AI model.
  • Web Content Interaction: Chat directly with the content of a webpage to get explanations or analysis.
  • Multi-Browser Support: Works on Chrome, Brave, Edge, Firefox, and other major browsers.
  • Keyboard Shortcuts: Quickly open the sidebar or web UI using customizable shortcuts.
  • Local AI Providers: Supports Ollama, Chrome AI (Gemini Nano), and other local AI providers.
  • Document Parsing: Chat with PDFs, CSVs, and other document formats.
  • Offline Compatibility: Designed for offline use, allowing you to run AI models locally.
  • Privacy-Focused: All interactions happen locally, with no personal data collection. Data is stored in your browser’s local storage.
  • Developer Mode: Includes tools for developers to test and extend the extension.

Installation and Usage

You can install Page Assist directly from the Chrome Web Store or Firefox Add-ons Store, or manually install it.

  1. Visit the Chrome Web Store: https://chromewebstore.google.com/ and search for Page Assist.
    • Or use this direct link: Page Assist on Chrome Web Store.
  2. Click Add to Browser to install the extension.
  3. Once installed, click the extension icon in the top-right corner of your browser to open it.
  4. If Ollama isn’t running, you’ll be prompted to start it.
  5. Select your preferred AI model and enter your prompt to start chatting.
  6. If the interface is in English and you prefer another language, click the gear icon in the top-right corner to switch to your preferred language (e.g., Chinese).
  7. You can also customize the default search engine for the extension’s web search feature.

GitHub Repository

For developers or those interested in the source code, visit the Page Assist GitHub page:
https://github.com/n4ze3m/page-assist.
